## 图片

### 使用Markdown

使用[markdown语法](https://www.markdownguide.org/basic-syntax/#images)可以通过以下代码添加图片

```md
![标题](/path/image.jpg)
```

请注意，图片文件大小必须小于5MB。否则，我们建议将图片托管在[Cloudinary](https://cloudinary.com/)或[S3](https://aws.amazon.com/s3/)等服务上。然后您可以使用该URL进行嵌入。

### 使用嵌入

要获得更多的图片自定义选项，您也可以使用[嵌入](/writing-content/embed)来添加图片

```html
<img height="200" src="/path/image.jpg" />
```

### iframe框架

在文档中加载另一个HTML页面。最常用于嵌入视频。

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
